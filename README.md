# Netflix-MERN
Netflix clone using MERN + Redux for state management. Features user auth, video streaming, and responsive design. Includes Express backend, MongoDB for data, and React for UI.

This project is an advanced full-stack clone of Netflix, leveraging the power of the MERN technology stack (MongoDB, Express.js, React.js, Node.js) along with Redux for state management, to build a feature-rich, scalable web application. This repository houses the complete source code for a complex version of Netflix, focusing on user authentication, interactive video streaming, personalized user experiences, and real-world application functionalities.

Core Features:

Dynamic User Authentication: Implements secure signup and login with JWT and Express middleware for session management and authentication across routes.
Interactive Video Streaming: Utilizes advanced React components for a fluid, buffer-free video streaming experience, akin to commercial streaming platforms.
State Management with Redux: Leverages Redux alongside React for efficient global state management, ensuring a consistent and responsive user interface.
Responsive and Adaptive Design: Crafted with mobile-first design principles using React-Responsive and CSS Flexbox/Grid, offering an unparalleled viewing experience across all devices.
CRUD Operations with MongoDB: Models for user data, video metadata, and preferences are designed with MongoDB schemas, enabling comprehensive CRUD operations through Express.js routes.
Advanced Search and Recommendation Engine: Integrates a custom search and recommendation algorithm, improving content discoverability and user engagement.
Technologies and Libraries:

Frontend: React.js for building a dynamic UI, Redux for global state management, React Router for navigation, and Axios for making HTTP requests.
Backend: Node.js with Express.js framework for RESTful API development, Mongoose for MongoDB object modeling, and bcrypt for password hashing.
Database: MongoDB for a flexible, scalable document database that stores user profiles, video content, and interactions.
Authentication and Security: Uses JWT for secure authentication, along with Express middleware for route protection and Cross-Origin Resource Sharing (CORS) for security.
